# goyoubbs

goyoubbs is an open source web forum built on Golang, fasthttp and leveldb.

Demo online https://youbbs.org/

## Usage

Require go 1.19+

Download source code and build.

```
git clone https://github.com/ego008/goyoubbs
cd goyoubbs
go build .
./goyoubbs
```
